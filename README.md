# 📊 Sales Analytics Project (SQL + Power BI)

## 🇺🇸 Project Overview
This project performs an end-to-end sales data analysis using SQL and Power BI, focusing on revenue performance, customer behavior, and business insights.

## 🇧🇷 Visão do Projeto
Este projeto realiza uma análise completa de vendas utilizando SQL e Power BI, com foco em desempenho de receita, comportamento de clientes e geração de insights para o negócio.

---

## 🎯 Objectives
- Analyze sales performance over time  
- Identify top-performing products  
- Understand customer purchasing behavior  
- Generate actionable business insights  

---

## 🛠️ Tools & Technologies
